# Zerega Arquitectos Ltda - version 1

Sitio web estatico para Zerega Arquitectos Ltda, listo para abrir en navegador, subir a GitHub o desplegar en Vercel.

## Archivos principales

- `index.html`: estructura y contenido del sitio.
- `styles.css`: diseño responsive.
- `script.js`: menu movil y comportamiento del encabezado.
- `assets/`: logo e imagenes reales usadas en la pagina.
- `vercel.json`: configuracion simple para Vercel.

## Despliegue

En Vercel, importar esta carpeta como proyecto estatico. No requiere build.

Para revisar localmente:

```bash
python -m http.server 5173
```

Luego abrir `http://localhost:5173`.
